Game tester adalah profesional yang bertanggung jawab untuk memainkan dan menguji game selama proses pengembangan
untuk menemukan dan melaporkan bug, glitch, dan masalah lain yang dapat mempengaruhi pengalaman pengguna.

Tugas Utama Game Tester

  1. Memainkan Game Secara Menyeluruh: Memainkan game dari awal hingga akhir, mencoba setiap fitur dan mode permainan.
  2. Mendeteksi Bug dan Glitch: Mengidentifikasi dan mereproduksi bug, glitch, dan masalah teknis lainnya.
  3. Melaporkan Masalah: Menulis laporan bug yang detail, termasuk langkah-langkah untuk mereproduksi
    masalah, tangkapan layar, atau rekaman video.
  4. Uji Fungsionalitas: Memastikan semua fitur game berfungsi sesuai spesifikasi desain.
  5. Uji Kinerja: Mengevaluasi kinerja game, termasuk frame rate, waktu muat, dan penggunaan sumber daya.
  6. Uji Kepatuhan: Memastikan game memenuhi standar dan pedoman platform tertentu (seperti konsol atau toko aplikasi).
  7. Uji Pengalaman Pengguna: Menilai aspek gameplay dan antarmuka pengguna untuk memastikan pengalaman
    yang menyenangkan dan intuitif.

Jenis Pengujian Game

  1. Pengujian Fungsional: Memastikan bahwa semua fitur dan mekanik game berfungsi dengan baik.
  2. Pengujian Kompatibilitas: Memastikan game berjalan dengan baik di berbagai perangkat keras dan konfigurasi perangkat lunak.
  3. Pengujian Kinerja: Menilai performa game dalam hal frame rate, waktu muat, dan stabilitas.
  4. Pengujian Pengalaman Pengguna (UX): Mengevaluasi kenyamanan dan kesenangan pemain saat bermain.
  5. Pengujian Keamanan: Mendeteksi potensi eksploitasi dan masalah keamanan dalam game.
  6. Pengujian Kepatuhan (Compliance Testing): Memastikan game memenuhi persyaratan dan pedoman dari platform
    distribusi (seperti PlayStation, Xbox, Steam).
